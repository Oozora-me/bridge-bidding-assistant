# 二盖一进局逼叫 (2/1 Game Forcing) 体系完整说明

## 第一章：体系基础与核心定义

### 1.1 体系名称

二盖一进局逼叫 (Two-Over-One Game Forcing)，简称 2/1 GF。

### 1.2 核心机制

当应叫方在第一或第二家开叫的 1♥ 或 1♠ 后，在二阶叫出新花色（非跳叫、非加叫），该叫品**自动建立进局逼叫**关系。双方此后必须将叫牌推进到成局定约（3NT、4♥、4♠、5♣、5♦）或更高，除非有特殊约定允许提前停叫。

### 1.3 触发条件（必须同时满足）

1. 开叫方为第一或第二家位置。
2. 开叫花色为 1♥ 或 1♠。
3. 对方未进行任何叫牌（无加倍、无争叫）。
4. 应叫方是未派司过的牌手。

### 1.4 不适用 2/1 GF 的情况

- 低花开叫后的二阶应叫（如 1♦ - 2♣ 通常仅为邀请，非 GF）。
- 第三、四家开叫后。
- 对方干扰（争叫或加倍）后。
- 应叫方已派司过。

## 第三章：无将开叫及其后续

### 3.1 1NT 开叫 (15-17 HCP)

#### 应叫结构

- **2♣**：斯台曼 (Stayman)，问高花4张。答叫：2♦ 无4张高花；2♥ 有4张红心；2♠ 有4张黑桃且无4张红心。
- **2♦**：雅各比转移叫至 2♥，保证5+张红心。
- **2♥**：雅各比转移叫至 2♠，保证5+张黑桃。
- **2♠**：要求转移至 3♣（低花转移的起点）。
- **2NT**：自然邀请，8-9 HCP，无4张高花。
- **3♣ / 3♦**：自然邀请，6张套。
- **3♥ / 3♠**：自然逼局，通常为 5-5 以上双高花，斯莫伦 (Smolen) 的另一种表达。
- **4♣**：罗马葛伯关键张问叫，前提是已通过斯台曼找到高花配合。
- **4♦**：德克萨斯转移叫至 4♥。
- **4♥**：德克萨斯转移叫至 4♠。
- **4NT**：满贯邀叫，无4张高花。

#### 对方干扰后（莱本索尔约定）

- 对方自然争叫 2X 后：
  - 加倍 = 惩罚性。
  - 二阶新花 = 不逼叫。
  - 三阶新花 = 逼局。
  - 2NT = 莱本索尔接力至 3♣，区分强弱止叫或有无止张。

### 3.2 2NT 开叫 (20-21 HCP)

应叫结构与 1NT 开叫后相似，使用：

- 3♣ = 斯台曼。
- 3♦ / 3♥ = 雅各比转移叫至相应高花。
- 3♠ = 要求转移至 4♣。
- 4♣ = 罗马葛伯关键张问叫（高花配合后）。
- 4♦ / 4♥ = 德克萨斯转移叫。

## 第四章：一阶高花开叫与 2/1 GF 核心流程

### 4.1 一阶高花开叫条件

开叫 1♥ 或 1♠ 保证 **5张以上套**，牌力第一、二家 **12-21 HCP**。第三家可放宽至 10+ HCP 且有较好套。

### 4.2 2/1 应叫品列表（仅此六种为 GF）

- 1♥ 开叫后：2♣、2♦
- 1♠ 开叫后：2♣、2♦、2♥

### 4.3 2/1 GF 建立后的叫牌原则

#### 4.3.1 速达原则 (Principle of Fast Arrival)

- 在 GF 已建立的情况下，**直接跳叫到成局定约** 表示低限牌力（约 12-14 HCP），无满贯兴趣。
- **在成局水平以下缓慢叫牌**（如三阶叫回将牌）表示有额外牌力（15+ HCP），暗示满贯可能。

#### 4.3.2 严肃 3NT (Serious 3NT)

- 适用场景：GF 已建立，且在三阶确认 8 张以上高花配合。
- 叫品含义：叫 3NT 是**人为约定**，表示**严肃的满贯兴趣**，要求同伴开始扣叫。
- 非严肃 3NT 变体：3NT 表示轻度兴趣，直接扣叫表示严肃兴趣。需搭档事先约定。

#### 4.3.3 开叫人再叫的含义

- **再叫原高花 (2M)**：6+张套，牌力 12-18 HCP。
- **二阶新花**：自然，4+张第二套，12-18 HCP。
- **2NT**：均型，通常 12-14 或 18-19 HCP（后续需区分）。
- **三阶新花（高逆叫）**：5-5 以上两套，额外牌力。
- **跳叫原高花 (3M)**：6+张好套，约 15-18 HCP，确立将牌并邀请扣叫。

#### 4.3.4 应叫人再叫的含义

- **再叫原 2/1 花色**：在某些变体中，此叫品**取消 GF**，变为邀叫牌力（需搭档约定）。
- **加叫开叫人高花至三阶**：有 3 张支持，额外牌力，满贯试探。
- **叫 3NT**：均型低限（或按约定为额外牌力），建议最终定约。
- **第四花色**：人为逼局，询问档张或进一步牌型。

### 4.4 逼叫性 1NT (Forcing 1NT)

#### 4.4.1 定义与条件

- 应叫方持 **6-12 HCP**，对同伴高花无 3 张支持，且不够 2/1 GF 牌力时，应叫 **1NT**。
- 该 1NT 对开叫人是**一轮逼叫**，开叫人不能 Pass。

#### 4.4.2 开叫人再叫选择

1. 叫出 4 张以上第二套。
2. 三张低花过渡（如 1♠ - 1NT - 2♣ 可能仅 3 张）。
3. 再叫 6 张以上原高花。
4. 跳叫新花表示 5-5 以上且强牌。
5. 叫 2NT 表示 18-19 HCP 均型。

#### 4.4.3 半逼叫 1NT 变体

部分搭档约定，开叫人持 5-3-3-2 且 12-13 HCP 的低限均型时，可以选择 Pass 1NT。此为可选约定。

## 第五章：一阶高花开叫的其他应叫与加叫

### 5.1 简单加叫 (1M - 2M)

- 6-10 HCP，3 张以上支持，不逼叫。

### 5.2 限制性加叫 (1M - 3M)

- 10-12 HCP，4 张以上支持（或好的 3 张），邀请进局。

### 5.3 杰可贝 2NT (Jacoby 2NT)

- 应叫 2NT 表示 4+张支持，进局逼叫（12+ HCP）。
- 开叫人答叫：
  - 3♣ / 3♦ / 3♥ / 3♠：所叫花色为单缺（Splinter）。
  - 3M：强牌（18+ HCP），无单缺。
  - 4M：低限（12-14 HCP），无单缺。
  - 3NT：中限（15-17 HCP），无单缺。

### 5.4 伯根加叫 (Bergen Raises)

- 1M 开叫后：
  - 3♣：4+张支持，7-10 HCP（建设性加叫）。
  - 3♦：4+张支持，10-12 HCP（限制性加叫）。
  - 3M：4+张支持，0-6 HCP（阻击性加叫）。

### 5.5 斯普林特 (Splinter)

- 双跳新花（如 1♠ - 4♣）表示对开叫高花有 4+张支持，所叫花色为单张或缺门，牌力约 11-15 HCP。

### 5.6 跳叫新花色（单跳）

- 自然叫，表示 6-7 张好套，9-12 HCP，邀请牌力，不逼叫。

## 第六章：一阶低花开叫及其后续

### 6.1 低花开叫原则

- 开叫 1♣ / 1♦ 保证 3 张以上。
- 4-4 低花时开叫 1♦；3-3 低花时开叫 1♣。
- 4-4-4-1 型且两门低花时，开叫 1♣。

### 6.2 低花开叫后的应叫

| 应叫         | 含义       | 牌力/牌型                           |
| :----------- | :--------- | :---------------------------------- |
| 1♦ / 1♥ / 1♠ | 自然，4+张 | 6+ HCP，逼叫一轮                    |
| 1NT          | 均型       | 1♣后 8-10 HCP；1♦后 6-10 HCP        |
| 2♣ / 2♦      | 反加叫     | 10+ HCP，5+张支持，逼叫至三阶该花色 |
| 2NT          | 均型邀请   | 1♣后 13+ HCP；1♦后 11-12 HCP        |
| 3♣ / 3♦      | 阻击性加叫 | 弱牌，5+张支持                      |
| 跳新花       | 强牌       | 16+ HCP，5+张好套                   |

### 6.3 沃尔什应叫方式 (Walsh)

- 应叫方持 6+ HCP 且有 4 张高花时，优先应叫高花，即使有更长低花套。
- 开叫人持均型牌时，再叫 1NT 可能隐藏 4 张高花。

### 6.4 新低花逼叫 (New Minor Forcing)

- 在 1m - 1M - 1NT 后，应叫人叫 **2♣**（若开叫 1♣ 则 2♦ 是新低花）是人为逼叫一轮。
- 要求开叫人进一步描述牌型：优先显示对同伴高花的 3 张支持，或显示另一高花 4 张套，否则叫回原低花或 NT。

### 6.5 第四花色逼叫进局 (Fourth Suit Forcing)

- 在非 GF 进程中，叫出第四花色（如 1♦ - 1♠ - 2♣ - 2♥）是人为进局逼叫。
- 不承诺所叫花色长度，要求开叫人描述止张或额外牌型。

## 第七章：高阶开叫与特殊开叫

### 7.1 2♣ 强开叫

- 均型 22+ HCP，或非均型至少 9 赢墩。
- 应叫：
  - 2♦：等待叫，示弱（0-7 HCP）或均型无好套。
  - 2♥ / 2♠ / 3♣ / 3♦：自然积极应叫，8+ HCP，5+张套，至少两张大牌。
  - 2NT：均型 8+ HCP。

### 7.2 弱二开叫 (2♦ / 2♥ / 2♠)

- 6 张套，5-11 HCP，主要集中在该套。
- 应叫：
  - 2NT：奥古斯特 (Ogust) 问叫，询问套质量和牌力。
  - 新花：逼叫一轮，自然。
  - 加叫：阻击性。

### 7.3 三阶阻击开叫

- 7 张套，遵循二三法则（局况不利时宕二，局况有利时宕三）。
- 应叫：新花逼叫，4♣ 为罗马关键张问叫（若花色配合）。

### 7.4 3NT 赌博性开叫

- 坚固 7 张低花套（AKQxxxx 或更好），边花无 A 或 K。
- 应叫人可 Pass，或叫 4♣（Pass or Correct），或叫 5♣/5♦（止叫）。

### 7.5 纳姆亚茨 (Namyats)

- 4♣ 开叫 = 强 4♥ 阻击，4♦ 开叫 = 强 4♠ 阻击。
- 表示约 8.5-9 赢墩，比直接开叫 4M 更强，允许同伴试探满贯。

## 第八章：满贯叫牌约定

### 8.1 罗马关键张黑木 (Roman Key Card Blackwood, RKCB)

- 前提：明确将牌花色。
- 问叫：4NT。
- 答叫（将牌 K 计为第五个关键张）：
  - 5♣ = 0 或 3 个关键张
  - 5♦ = 1 或 4 个关键张
  - 5♥ = 2 或 5 个关键张，无将牌 Q
  - 5♠ = 2 或 5 个关键张，有将牌 Q
- 后续询问将牌 Q：在 5♣/5♦ 答叫后，问叫人叫**最经济的新花**（不含将牌花色），答叫人加一级无 Q，加二级有 Q。
- 5NT 问 K：表示联手已有全部 5 个关键张，问边花 K 数量（不含将牌 K）。答叫 6♣=0，6♦=1，6♥=2，6♠=3。
- 特定花色 K 问叫：直接叫出该花色询问。

### 8.2 罗马葛伯关键张 (Roman Key Card Gerber)

- 前提：在 1NT 或 2NT 开叫后，通过斯台曼找到高花配合。
- 问叫：4♣。
- 答叫分级与 RKCB 相同（4♦=0/3，4♥=1/4，4♠=2/5 无 Q，4NT=2/5 有 Q）。

### 8.3 扣叫 (Cue-Bidding)

- 确认将牌配合后，在三阶或四阶叫出新花色表示第一轮或第二轮控制。
- 一般先扣叫第一轮控制（A 或缺门），遵循由低到高原则。
- 越过某一花色通常否认该花色控制。

### 8.4 自愿性五阶高花加叫

- 在满贯试探过程中，直接加叫到五阶高花（如 1♠ - 2♦ - 3♠ - 5♠）表示有额外实力但缺乏合适的扣叫手段，邀请同伴在牌力合适时叫到满贯。

## 第九章：竞争叫牌与防守叫牌

### 9.1 对方技术性加倍后

- 再加倍：10+ HCP，通常暗示对加倍花色有实力。
- 一阶新花：逼叫一轮。
- 二阶新花：不逼叫。
- 跳加叫：阻击性。
- 2NT：对开叫花色的限制性加叫或更强（乔丹 2NT）。

### 9.2 对方花色争叫后

- 扣叫对方花色：表示限制性加叫或更强，逼叫一轮。
- 负加倍 (Negative Double)：至 4♦ 水平，表示未叫高花套或额外牌力。
- 二阶新花：逼叫一轮。
- 三阶新花：逼局。
- 加叫：竞争性，6-10 HCP。

### 9.3 平衡位置叫牌

- 平衡位置加倍或叫牌，牌力可适当降低（约 2-3 HCP）。
- 应叫方需相应调整评估。

## 第十章：体系原则与推理要点总结

1. **触发 2/1 GF 即锁定成局**，后续叫牌以探索最佳定约和满贯为目标。
2. **牌型优先于牌点**：GF 后优先描述分布，速达原则区分牌力区间。
3. **人为约定叫均有严格触发条件**，必须结合进程准确识别。
4. **1NT 应叫在高花开叫后为逼叫**，是处理 6-12 HCP 无配合牌的主要手段。
5. **低花进程独立于 2/1 GF**，使用新低花逼叫和第四花色逼局等工具。
6. **竞争叫牌后 GF 失效**，需切换至竞争叫牌逻辑。
